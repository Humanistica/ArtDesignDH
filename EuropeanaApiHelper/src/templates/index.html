Europeana Search API


Europeana Search API est une application web simple permettant de constituer des corpus
à partir des collections disponibles sur le portail d'Europeana.
Cette application utilise sur l'API d'Europeana et nécessite une clé publique.



Clé publique :

Une clé publique est nécessaire pour vous authentifier auprès d'Europeana.
Pour récupérer une clé, rendez-vous sur cette page.


Conseils d'utilisation :

  • Entrez un mot clés,

  • Choisissez une période limite pour filtrer les résulats,

  • Pour une recherche plus précise : compléter les champs "auteur", "date de création", "provider" ou
    encore le "type" de document
